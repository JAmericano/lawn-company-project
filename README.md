# Lawn Company Project

## Description
This is a Django web application for a lawn care company.

The goal of this project is to create a website where customers can:

- View lawn care services
- Request quotes
- Contact the company
- See pricing information

## Technologies Used
- Python
- Django
- HTML / CSS
- SQLite Database

## User Stories
As a customer, I want to be able to schedule an appointment on the Lawn Mowing website on a calender.



As the company owner, I want to be able to create accounts so users can securely input their info and for me to avoid scammers.
